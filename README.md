# Mini Audio Framework

A small C++ audio framework project for learning audio software development.

## Requirements

- CMake
- Ninja
- C++20 compiler
- Visual Studio 2026 Community on Windows

## Build

```bash
cmake --preset windows-msvc-ninja-debug
cmake --build --preset windows-msvc-ninja-debug
```

## Run tests

```bash
ctest --preset windows-msvc-ninja-debug
```
